Build Server

This is your private build server running in a Docker container.

It contains several components:

GitLab

This is where it begins: a dedicated server which stores your source code repository.

Jenkins

Jenkins will execute the build pipeline for you and deliver an artifact.

SonarQube

Each build execution will deliver reports on unit test coverage etcetera. You can view the results here.

Artifactory

After a succesful build, the resulting artefacts are published here for download and deployment.

The final app

At the end of the build the final application is deployed here for a final test by the end user (that's you!) What is it going to be, Ali Baba?
